# Audio Files

Place the following audio files in this directory:

- ambiance.mp3 - Dark ambient background music
- marco.mp3 - Voice saying "Marco!"
- polo.mp3 - Voice saying "Polo!"
- footsteps.mp3 - Footstep sounds
- heartbeat.mp3 - Heartbeat sound for tension
- jumpscare.mp3 - Sudden loud sound for scares

You can find free horror sound effects at:
- freesound.org
- zapsplat.com
- sonniss.com (GameAudioGDC)
